# 贪心算法

从初始位置开始，找到当前位置能跳到的最远位置，在在最远位置与当前位置范围之内，继续查找能跳到的最远位置。

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var jump = function(nums) {
  let end = 0,
    longest = 0,
    count = 0;
  for (let i = 0; i < nums.length - 1; i++) {
    // 注意此处设定为i<length-1
    // 根据题目要求，假定每一个组合都能够跳到最终位置，因此最后一个位置不需要计算在内
    longest = Math.max(longest, nums[i] + i);
    if (end === i) {
      count++;
      end = longest;
    }
  }
  return count;
};
```

**时间复杂度**：`O(n)`

**空间复杂度**：`O(1)`
